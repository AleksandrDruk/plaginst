# Starter Gutenberg Kit

Starter Gutenberg Kit (SGK) is a lightweight WordPress plugin that adds a curated set of Gutenberg section patterns and block styles for building landing pages quickly with core blocks only.

## Installation

1. Zip the `starter-gutenberg-kit` folder.
2. In WordPress admin, go to **Plugins → Add New → Upload Plugin**.
3. Upload the ZIP, install, and activate.

## How to use

- In the block editor, open **Patterns → SGK Sections**.
- Insert a section, then edit text and reorder sections as needed.
- Use block styles in the block sidebar to switch table, card, and button variants.

## Section classes

Each pattern root Group block includes a section type and variant class:

- Types: `sec-hero`, `sec-intro`, `sec-how`, `sec-top`, `sec-deposit`, `sec-withdraw`, `sec-proscons`, `sec-alternatives`, `sec-safety`, `sec-trust`, `sec-faq`, `sec-methodology`, `sec-termslimits`
- Variants: `sec-variant-a`, `sec-variant-b`, `sec-variant-c`

## Block styles

- Tables: `sgk-table-a`, `sgk-table-b`, `sgk-table-c`
- Cards (Group): `sgk-card-a`, `sgk-card-b`
- Buttons: `sgk-btn-a`, `sgk-btn-b`

## Recommended landing page order

Hero → Intro → How → Top → Deposit/Withdraw → Pros/Cons → Alternatives → Responsible → FAQ → Trust

## Variability rule

Mix different section variants and table styles to avoid repeating the same visual footprint across pages.
